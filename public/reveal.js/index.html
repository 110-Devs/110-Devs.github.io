Einführung in
Proophboard
&
Cody-Engine

Proophboard ist eine führende Plattform zur visuellen Modellierung von Event-getriebenen Systemen. Sie ermöglicht es verteilten Teams, Softwarelösungen effizient zu entwerfen und umzusetzen. Die Cody-Engine generiert aus diesen Modellen funktionsfähige Prototypen.

Projektauftrag

Die Cody-Engine soll durch die Integration einer künstlicher Intelligenz erweitert werden, um die Prototypen basierend auf Benutzerinteraktionen anzupassen zu können. Dadurch soll das Erscheinungsbild der generierten Anwendungen individuell an die spezifischen Anforderungen der Nutzer angepasst werden.

"Cody generates basic React UI with fully working backend integration.
AI is used to customize look and feel of UI, without breaking backend API integration."
